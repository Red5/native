# MPEG 1 Video and MP2 Audio

## Prerequisites

### Linux

SDL2 and OpenGL are needed:

```sh
sudo apt install libglm-dev libglew-dev libsdl2-dev
```

### Windows

[GLEW](https://www.opengl.org/sdk/libs/GLEW/)

[SDL2](https://www.libsdl.org/download-2.0.php)
